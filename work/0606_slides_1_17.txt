--- Slide 3 ---
Introduction
Future wireless systems require 
ultra-high spectral efficiency
 and the capability to support 
massive machine-type communications (mMTC)
 in highly dense networks.
Multiple access (MA) techniques are essential for 
improving spectral efficiency
 and accommodating a massive number of simultaneous connections in wireless networks.
Orthogonal multiple access (OMA) avoids inter-user interference(IUI) by assigning disjoint resources, its user capacity is fundamentally limited by the number of orthogonal resource units. 
Non-orthogonal multiple access (NOMA) improves resource utilization by allowing multiple users to share the same resource.
In uplink NOMA, the receiver structure is often the key factor that determines both reliability and practicality. Different NOMA schemes rely on different user-separation principles.


--- Slide 4 ---
Introduction of Multiple Access
Multiple Access(MA)
 refers to methods for allowing multiple users to share communication resources.
OMA (Orthogonal Multiple Access)
Users occupy separate orthogonal resources (time, frequency, code).
No inter-user interference
 →
 Single user detector(SUD) is used.
Lower spectral efficiency in multiple user system.
NOMA(Non orthogonal Multiple Access)
Users  simultaneously access the same resource.
Have inter-user interference 
→ 
multiple user detector(MUD) is used.
Higher spectral efficiency, supports more users.
Lower transmission latency (no scheduling request from users to base station is required).
A resource can be a frequency band, a timeslot, a signature or a subcarrier.

Power Domain-NOMA(PD-NOMA) system: the receiver employs the 
power levels
 associated to users to separate users.
